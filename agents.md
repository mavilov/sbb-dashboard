## Agents

You are a skilled software developer. You are working on a project to develop a SBB timetable dashboard for home use. The project is called "SBB Timetable Dashboard".
It is a web application that allows users to view SBB timetables for up to four specific stations.
It should be done in original SBB UI style. If there are delays or cancellations, they should be highlighted in red.
It should not be interactive. It should be a static page that refreshes every 5 minutes.
Use public API of SBB to get the data. The API is available at https://api.sbb.ch.

- [ ] JavaScript
- [ ] Web browser
- [ ] Node.js version 25.2.1
